# Livestock Booking Platform

## Purpose

A web application that allows users to browse and book livestock animals for Qurbani. Users can explore available animals, view details, and submit booking requests with secure authentication.

## Live URL

[https://a-8-livestock-booking-platform.vercel.app](https://a-8-livestock-booking-platform.vercel.app)

## Key Features

- Email and password registration and login
- Google OAuth sign in
- Browse all livestock with filtering and sorting
- Animal detail page with full information
- Booking form for selected animals
- User profile view and update
- Protected routes for authenticated users
- Responsive design for all devices
